Problem: For what real values of $x$ is $$\sqrt{x + \sqrt{2x-1}} + \sqrt{x - \sqrt{2x-1}} = A$$ given 

\begin{enumerate}
\item $A = \sqrt{2}$
\item $A = 1$
\item $A = 2$
\end{enumerate}

where only non-negative real numbers are admitted for square roots? (Source: 1959 IMO)

Solution: We can simplify this equation by squaring both sides.

\begin{align*}
\sqrt{x + \sqrt{2x-1}} + \sqrt{x - \sqrt{2x-1}} &= A \\
\left(\sqrt{x + \sqrt{2x-1}} + \sqrt{x - \sqrt{2x-1}}\right)^2 &= A^2 \\
x + \sqrt{2x-1} + \sqrt{x^2 - (2x - 1)} + \sqrt{x^2 - (2x - 1)} + x - \sqrt{2x-1} = A^2 \\
2x + 2\sqrt{x^2 - (2x - 1)} = A^2 \\
2x + 2\sqrt{x^2 - 2x + 1} = A^2 \\
2x + 2\sqrt{(x-1)^2} = A^2 \\
2x + 2(x-1) = A^2 \\
4x - 2 &= A^2 \\
4x &= A^2 + 2 \\
x &= \frac{A^2 + 2}{4}
\end{align*}

When $A = \sqrt{2}$, we have $x = 1$. When $A = 2$, we have $x = \dfrac{3}{2}$.

It looks like $A = 1 \implies x = \dfrac{3}{4}$. But plugging $x = \dfrac{3}{4}$ into the original equation does not give us $A = 1$. So I think there is no solution for $A = 1$.

Thus the solutions are $x = \left\{1, \dfrac{3}{2} \right\}$.
